A Flixster widget for Wordpress

== description ==	

Flixster is a social web site that can be integrated as a Facebook application. It lets you rate movies and share reviews. This widget can display the last reviews and ratings a Flixster user released.

== Installation ==

1. Upload the full directory into your wp-content/plugins directory.
2. Activate the widget in the Plugins tab of the administration interface. 
3. Add it to the sidebar in Presentation -> Widgets. 
4. Configure it with a Flixster ID.
